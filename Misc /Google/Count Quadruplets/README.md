[**Problem Statement:**](https://www.desiqna.in/13378/google-oa-sde2-march-2022)

Given an array `nums` of size `N`, find the number of quadruplets such that `i < j < k < l` and `nums[i] < nums[j] < nums[k] < nums[l]`.

**Constraints:**

- `1 <= N <= 1000`
- `1 <= nums[i] <= 100000000`
